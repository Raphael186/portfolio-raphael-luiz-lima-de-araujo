# Traduzindo Lógica para Python

## Descrição do Projeto

[cite_start]Este projeto reúne uma série de algoritmos práticos desenvolvidos para converter conceitos de lógica de programação em implementações funcionais utilizando a linguagem Python[cite: 8, 26]. [cite_start]O objetivo principal é exercitar o uso de estruturas de repetição, condicionais compostas e manipulação de tipos de dados para resolver problemas do quotidiano, como gestão de vendas, monitorização climática e cálculos financeiros[cite: 8, 26].

[cite_start]As atividades foram estruturadas de forma modular, onde cada função resolve um problema específico com entradas dinâmicas do utilizador[cite: 8, 26].

## Tecnologias e Lógica Utilizada

* [cite_start]**Linguagem:** Python 3.10 [cite: 8, 26]
* [cite_start]**Conceitos Aplicados:** Estruturas de repetição (`for`), estruturas de decisão (`if/elif/else`), tratamento de números flutuantes e funções modulares[cite: 8, 26].
* [cite_start]**Ambiente de Desenvolvimento:** Google Colab[cite: 8].

## Estrutura das Atividades

O sistema foi dividido em quatro módulos principais, cada um focado em uma lógica específica:

1. [cite_start]**Processamento de Vendas:** Calcula o total de uma compra com base em múltiplos produtos, aplicando descontos progressivos de 5% ou 10% dependendo do valor total atingido[cite: 8, 26].
2. [cite_start]**Análise de Clima:** Regista as temperaturas de uma semana inteira, calcula a média e emite alertas de condições extremas para temperaturas acima de 45°C ou abaixo de -5°C[cite: 8, 26].
3. [cite_start]**Sistema de Notas da Turma:** Gerencia o registo de notas de alunos, calculando médias individuais e classificando-os entre "Aprovado", "Recuperação" ou "Reprovado"[cite: 8, 26].
4. [cite_start]**Simulador de Poupança:** Simula o crescimento de um investimento mês a mês, considerando aportes mensais e juros compostos, com um alerta ao atingir a meta de 10.000,00[cite: 8, 26].

## Ficheiros do Projeto

* [cite_start]`atividades_desenvolvidas.py`: Ficheiro contendo todas as funções de lógica traduzidas para Python[cite: 8].

## Exemplos de Fluxo Lógico

* [cite_start]**Vendas:** Se o total da compra for R$ 600,00, o algoritmo identifica que é superior a R$ 500,00 e aplica automaticamente o desconto de 10%[cite: 8, 26].
* [cite_start]**Clima:** Se qualquer temperatura lida for 46°C, o sistema altera o estado do alerta para verdadeiro e exibe um aviso de segurança no final do processamento[cite: 8, 26].
* [cite_start]**Educação:** Alunos com média entre 5.0 e 6.9 são automaticamente encaminhados para o estado de "Recuperação"[cite: 8, 26].

## Como Executar

1. [cite_start]Aceda ao ambiente do Google Colab ou utilize um interpretador Python local[cite: 8].
2. [cite_start]Carregue o ficheiro `atividades_desenvolvidas.py`[cite: 8].
3. [cite_start]Execute a célula de código e siga as instruções no terminal para introduzir os dados solicitados por cada função[cite: 8].
